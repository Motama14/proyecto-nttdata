Proyecto final: zoológico

Esta aplicación permite gestionar cuidadores y animales, usando una relación 1:M: un cuidador puede tener muchos animales asignados.

Desde el frontend podemos listar cuidadores, ver el detalle de cada uno, consultar sus animales y crear nuevos registros mediante formularios reactivos.

Ver cuidadores

Backend

Spring Boot, API REST, JPA, H2 en memoria y relación entre dos tablas.

Frontend

Angular con componentes standalone, signals, observables, formularios reactivos y rutas.
